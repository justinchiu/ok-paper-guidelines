# ok-paper-guidelines

A rough transcription of Sasha's [How to write an okay research paper](https://www.youtube.com/watch?v=qNlwVGxkG7Q) video.
Adapted for using with agents.

## Files

- `PAPER_GUIDELINES.md`: cleaned, agent-friendly guidelines (section-by-section).
- `PAPER_GUIDELINES.md`: agent instructions / section playbook.
- `keypoints.md`: one-page checklist / quick reference.
- `transcription.txt`: rough full transcription source.
